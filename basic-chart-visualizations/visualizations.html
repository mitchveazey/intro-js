The focus of this assignment is to draw simple SVG elements on a webpage and build simple visualizations from them.

These visualizations should be created using HTML, CSS, and SVG.
The webpage will have four charts: a single bar, one bar chart, one line chart, and one pie chart.
Each chart should be drawn in an svg element of size 400 x 400 pixels.
The layout is not important -- stack these vertically on top of each other.

One red bar

Create a single bar with the height of 250. The color of the bar should be red.

Green bar chart

Draw a bar chart. Your bar chart should take all 400 pixels in the SVG. The individual bars should have the following height, in order: 150, 225, 225, 300, 300, 225, 225, 150.

Line chart

Draw a line chart composed of the SVG line element. The heights of the vertices in the polyline should be the same as the heights of the individual bars for your bar chart.

Pie chart

Draw a pie chart with two wedges. The first wedge will span 90 degrees and will be green, and the second wedge will span the remaining 270 degrees, and will be yellow.
